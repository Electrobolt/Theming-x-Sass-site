CONTROL DIRECTIVES AND EXPRESSIONS


Sass control directives and expressions facilitate you to use styling based on some conditions.

@if Directive

@if directive specifies the execution of code statements according to the result of the expression. A @if statement can be used with several @if else statements and one @else statement.

@else-if Directive

@else-if directive is used when @if directive is failed. If it is failed @else directive is used.

@for Directive

@for directive facilitates you to generate a style in a loop. It is used when you require repeatedly set of styles. It uses counter variable to set the output for each iteration.

@each Directive

@each directive contains the value of each item in the list. It is also used with multiple assignments and multiple assignments with map.

@each directive with multiple assignments

We can use @each directive with multiple values like $var1, $var2, $var3, ... in .

@for Directive with To Keyword

@for directive, the "to" keyword is used to specify the range form value to the value before value.

@while Directive

The @while directive is very similar to @for directive. It also takes SassScript expressions and iteratively outputs nested styles until the statement returns to false. The value of counter is changed (increased/ decreased) after every iteration.
