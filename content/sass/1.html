WHAT IS SASS?

SASS stands for Systematically Awesome Style Sheets.

Sass is an extension of CSS. It is also known as CSS pre-processor.

It is used to add power and elegance to the basic language. It facilitates you to add variables, nested rules, mixins, inline imports, inheritance and more, all with fully CSS-compatible syntax.
It is very useful to handle large style sheets by keeping them well organized and running quickly small style sheets.
